# Installation

## Requirements

- PHP 8.2 or higher
- Composer

## Install via Composer

Install the library using Composer:

```bash
composer require byjg/authuser
```

## Running Tests

Because this project uses PHP Session, you need to run the unit tests with the `--stderr` flag:

```bash
./vendor/bin/phpunit --stderr
```
